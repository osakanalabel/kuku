# ポケ●ン九九

九九の暗記練習アプリです。正解するとポケ●ンをゲットできます。

🔗 **[あそんでみる](https://osakanalabel.github.io/kuku/)**

![QRコード](https://osakanalabel.github.io/kuku/qr.png)

## 特徴

- スマホブラウザ対応（縦画面固定）
- 経験値・レベルアップシステム
- ポケ●ンボックス（全1025匹）
- ログインボーナス・連続ログインストリーク
- データはブラウザに保存（IndexedDB）
- キャラクター画像は一切表示されません

## 技術

- Vue 3 (CDN) + IndexedDB — `index.html` 単体で動作

## その他

- 本アプリは個人の学習・研究を目的としています。
